About Y-SQLVIZ

Y-SQLVIZ is designed as an educational learning platform to help students and developers understand SQL JOIN operations, set operations, and execution order through interactive visualization. Whether you're learning SQL for the first time or teaching others, this tool is built for clarity and understanding.
