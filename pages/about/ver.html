Version History

Changelog

A record of every release of SQL Visualizer — what changed, what was added, and what was fixed. Versions are listed most recent first.

v1.2

March 2026
Latest
Major Release - Late March

A substantial overhaul across the board. The SQL engine was rewritten from scratch to handle joins and set operators reliably, the Generated SQL panel was introduced with live output, and the Execution Order visualizer was added as a new standalone page.
The overall visual language was refreshed and the page structure simplified throughout.

  • Introduced the Generated SQL panel. A live code block that reflects the current join configuration, with a one-click copy button.
  • Rewrote the SQL engine to support JOINS and SET OPERATORS
  • Added two pre-loaded sample datasets in pages [01] covering common patterns.
  • New Execution Order page with an 12-step expandable timeline
  • Sidebar groups now labelled and separated by section
  • Refreshed typography and theme overall.

v1.1

March 2026
Latest
minor Release - Early March

Fleshed out the Joins section with live Venn diagrams and a result preview table generated from sample data.

  • Interactive SVG Venn diagram updates per join type selection
  • Result preview renders up to seven rows from the in-memory dataset
  • Added CROSS JOIN and SELF JOIN coverage
  • SQL syntax highlighting applied across all code blocks

v1.0

Latest

First implementaion. A single page HTML file covering the four core join types with only two maually added tables with only one field and static venn diagrams and static results.
