Version History

Changelog

A record of every release of SQL Visualizer — what changed, what was added, and what was fixed. Versions are listed most recent first.

v1.3

March 2026
Latest
Major - Late March

A major overhaul focusing on the live Generated Sql and a full visual refresh. The page structure was simplified and the SQL engine was rewritten to handle joins and operators reliably.
Introduced the Execution Order page and improved the sidebar navigation with grouped sections and active-state highlighting.

  • Introduced the Generated Sql tab with a live SQL generation code block based on selected data. Copy button added to code blocks.
  • Rewrote the SQL engine to support JOINS and SET OPERATORS
  • Added two pre-loaded sample datasets in pages [01] covering common patterns.
  • New Execution Order page with an 12-step expandable timeline
  • Sidebar groups now labelled and separated by section
  • Refreshed typography and theme overall.

v1.2

March 2026

Fleshed out the Joins section with live Venn diagrams and a result preview table generated from sample data.

  • Interactive SVG Venn diagram updates per join type selection
  • Result preview renders up to seven rows from the in-memory dataset
  • Added CROSS JOIN and SELF JOIN coverage
  • SQL syntax highlighting applied across all code blocks

v1.0

February 2025
Initial release

First public release. A single HTML file covering the four core join types with static diagrams and annotated SQL examples.

  • Static diagrams for INNER, LEFT, RIGHT, and FULL OUTER joins
  • Annotated SQL syntax with keyword highlighting
  • Fixed top navigation and left sidebar with anchor links
  • Dark mode default with no theme switching
