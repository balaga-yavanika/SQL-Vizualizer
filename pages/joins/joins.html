A visual, step-by-step breakdown of working of Joins in SQL.

Before you start

Start with preset sample data or add tables dynamically using the + add table button. Create columns and sample data in each table.
Choose a join type: INNER (matching rows only), LEFT (all left + matches), RIGHT (all right + matches), FULL (all rows) + 6 more.
Try set operators: UNION (combine), INTERSECT (common rows), EXCEPT (difference).
Set join conditions to connect tables using keys. Watch the SVG diagram update. See results in the result table.
View the generated SQL.
Click Reset All to start over.

⚠️ Anti & Semi join SQL uses Spark SQL syntax (LEFT ANTI JOIN, LEFT SEMI JOIN) — not standard SQL. In PostgreSQL / MySQL, use NOT EXISTS / EXISTS subqueries instead.
